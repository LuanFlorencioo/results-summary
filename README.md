## Getting Started

First, install all deps:
```bash
npm install
# or
yarn
# or
pnpm install
```

Now run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
